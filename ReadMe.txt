Install:
--------

- If you are reading this you have already unzipped the folder named "TTCInstall".

- Inside this folder you will see a file named "header.php"
Replace your file http://yoursite.com/header.php with this new one.

- You will also see a folder named "TechThemeConfig"
place the folder in your modules folder
http://yoursite.com/modules/TechThemeConfig.

- There are 3 "TechThemes" included with this release.
 "TechDigiII"
 "TechKittinII"
 "TechThemeDefaultII"

Take these 3 folders and place them each in your themes folder.

http://yoursite.com/themes/TechDigiII/
http://yoursite.com/themes/TechKittinII/
http://yoursite.com/themes/TechThemeDefaultII/

- You may now login to your sites System Admin panel and install this module
as you would any other. Once installed, you may now configure your theme.

Note: Remember to enable your new themes.
SystemAdmin/preferences/General Settings
---------------------------------------------------------------------


Update:
-------

- If you already have version one installed,
simpley go to your system admin > modules and deactivate then uninstall
your current version.

- Now take your new TechThemeConfig folder and drop it in to your modules folder
and be sure to replace all exisiting files with the new ones.

http://yoursite.com/modules/TechThemeConfig

- Also in the TTCInstall folder you will see a new header.php
replace your current header.php with this new one.

http://yoursite.com/header.php

- As for the new themes drop the new folders in to your theme folder
and activate through your admin panel as you normally would any other theme.
